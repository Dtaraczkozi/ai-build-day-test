Project for AI Build Day.

## Project Doomscrolling

A screen time management app UI concept built with Next.js 16, React 19, Tailwind CSS v4, and Lucide React icons.

### Features

Three mobile screens implemented as phone mockups:

- **Home Dashboard** — Today's screen time, quick stats, app limits with progress bars, AI suggestions
- **Usage Analytics** — Weekly bar chart, top apps breakdown, weekly score
- **Mental Training** — Guided sessions, training categories, XP progress tracking

### Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
